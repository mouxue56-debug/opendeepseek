# Video Script Flywheel

Use this skill when the user wants to explain why OpenDeepSeek exists or persuade people to use DeepSeek V4 with Hermes Agent.

## Core Thesis

Do not position OpenDeepSeek as "another chat UI". Position it as:

```text
DeepSeek V4 API + Hermes Agent + Open WebUI
= cheap model intelligence becomes real personal automation.
```

## Narrative Structure

1. Hook:
   - "DeepSeek 官方聊天好用吗？能聊，但它不像一个真正的助手。"
   - "真正可惜的是：DeepSeek V4 的 Agent 能力很强，但大多数人只把它当聊天框。"

2. Pain:
   - No durable context.
   - Limited workflow.
   - Search/chat toggles are not automation.
   - It cannot create files, run tasks, schedule reminders, or work from your computer.

3. Solution:
   - Open WebUI gives a friendly web/mobile/PWA interface.
   - Hermes gives memory, skills, terminal/files, cron, subagents.
   - DeepSeek V4 gives affordable reasoning.

4. Flywheel:
   - User gives real tasks.
   - Agent uses tools and makes artifacts.
   - API usage creates feedback and demand.
   - Model and user both improve through repeated real-world attempts.

5. National/price angle:
   - The model is cheap enough to use heavily.
   - Supporting domestic models means using them in serious workflows, not just praising them.

6. Demo beats:
   - Desktop audit.
   - Generate a website.
   - Create a weekly report.
   - Create a reminder.
   - Use on phone through Tailscale/PWA.

7. CTA:
   - "不要只把 DeepSeek 当聊天软件，用它做事。"
   - "把 API 接到 Agent 上，才是真的开始。"

## Style

Use spoken Chinese. Keep it energetic, concrete, and slightly opinionated. Avoid corporate slogans.
