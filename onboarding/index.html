Open WebUI × Hermes Agent × DeepSeek V4 Flash

把 DeepSeek 变成真正能干活的本地 Agent。

OpenDeepSeek 不是又一个普通聊天壳子。它把 Open WebUI 的好用界面、Hermes 的文件/工具/定时任务能力、DeepSeek V4 Flash 的低成本推理接起来，让用户可以在网页、电脑和手机里直接布置真实任务。

四种模式

不用理解底层架构，按任务选择就行。

auto

自动模式

默认推荐。普通问题秒回；涉及文件、网页、图片、提醒和真实产物时自动切到 Hermes。

fast

极速问答

强制走 DeepSeek V4 Flash 轻量路径，适合解释、翻译、改文案和临时问答。

agent

真 Agent

强制 Hermes 执行。适合读取 /host、生成网页、写周报、整理文件、创建提醒。

deepwork

深度任务

给长报告、复杂代码、多步骤资料分析预留的高预算任务模式。

产物入口

为什么值得用

普通聊天走轻量路由，真正任务交给 Hermes。

能操作本机文件

Hermes 通过 /host 访问用户家目录，可以查看桌面、生成网页、写报告、整理资料。

能处理图片输入

Smart Bridge 先把图片本地落盘并 OCR，再把路径和摘要交给 Agent，不要求 DeepSeek API 直接吃图。

能做后台任务

适合周报、网页、PPT、资料搜集、定时提醒、长任务和多轮自动化，不只是陪聊。

文件权限

让 Agent 能干活，也要让用户心里有底。

OpenDeepSeek 的 Agent 能力来自 Hermes，所以它必须能看到你授权的本机目录。中国版默认会引导用户先使用专用目录，避免一上来把整个电脑交出去。

  • 默认输入目录：OpenDeepSeek-Inputs
  • 默认产物目录：OpenDeepSeek-Outputs
  • 默认记忆目录：OpenDeepSeek-Memory
  • 桌面、下载、文档需要用户明确授权后再开放
推荐默认 ~/OpenDeepSeek-Agent

这个目录适合录视频、试用和分享给小白。需要更强能力时，再在 .env 或后续 Portal 里扩大 `HERMES_HOST_DIR`。
