Open WebUI × Hermes Agent × DeepSeek V4 Flash

把 DeepSeek 变成真正能干活的本地 Agent。

OpenDeepSeek 不是又一个普通聊天壳子。它把 Open WebUI 的好用界面、Hermes 的文件/工具/定时任务能力、DeepSeek V4 Flash 的低成本推理接起来，让用户可以在网页、电脑和手机里直接布置真实任务。

真实任务 Hermes Agent 读文件、写网页、跑脚本、定时提醒

为什么值得用

普通聊天走轻量路由，真正任务交给 Hermes。

能操作本机文件

Hermes 通过 /host 访问用户家目录，可以查看桌面、生成网页、写报告、整理资料。

能处理图片输入

Smart Bridge 先把图片本地落盘并 OCR，再把路径和摘要交给 Agent，不要求 DeepSeek API 直接吃图。

能做后台任务

适合周报、网页、PPT、资料搜集、定时提醒、长任务和多轮自动化，不只是陪聊。

怎么运行

小白用一条命令，懂技术的可以手动启动。

推荐

一键安装

会自动拉取项目、打开配置向导、启动 Open WebUI + Smart Bridge + Hermes。

bash <(curl -fsSL https://raw.githubusercontent.com/mouxue56-debug/opendeepseek/main/install.sh)
本地仓库

已有代码时

进入项目目录后启动网页向导，粘贴 DeepSeek API Key 即可。

cd opendeepseek
./setup.sh --web
手动

Docker Compose

适合想自己编辑 .env、检查日志和控制容器的人。

cp .env.example .env
docker compose --profile full up -d
OD
