# 我应该下载哪个版本

OpenDeepSeek CN 会同时提供在线安装包和离线包。普通用户不要纠结技术名词，按自己的电脑和网络情况选就行。

## 最推荐

如果你能正常打开 Gitee，并且 Docker 可以拉国内镜像：

```bash
bash -c "$(curl -fsSL https://gitee.com/luoxueai/opendeepseek/raw/main/install-cn.sh)"
```

这会自动检测网络、创建 `.env`、启动 OpenDeepSeek，并默认只在本机开放 `http://localhost:3000`。

## 下载选择

| 你的设备 | 推荐文件 | 说明 |
|---|---|---|
| Mac Apple Silicon | `opendeepseek-cn-v0.5.0-cn-macos-arm64.zip` | M1/M2/M3/M4 系列芯片 |
| Mac Intel | `opendeepseek-cn-v0.5.0-cn-macos-amd64.zip` | 旧款 Intel Mac |
| Windows | `opendeepseek-cn-v0.5.0-cn-windows-amd64.zip` | 适合 Docker Desktop + WSL2 |
| Linux x86_64 | `opendeepseek-cn-v0.5.0-cn-linux-amd64.tar.gz` | 常见 PC / 服务器 |
| Linux ARM64 | `opendeepseek-cn-v0.5.0-cn-linux-arm64.tar.gz` | ARM 服务器 / 开发板 |

如果 Docker 镜像拉不下来，再额外下载：

| 镜像包 | 适用 |
|---|---|
| `opendeepseek-images-cn-v0.5.0-cn-amd64.tar.zst` | Intel/AMD 电脑 |
| `opendeepseek-images-cn-v0.5.0-cn-arm64.tar.zst` | Apple Silicon / ARM64 |

## 在线版还是离线版

在线版适合：

- Gitee 能打开
- Docker 可以拉国内镜像
- 只是自己电脑使用
- 想最快体验

离线版适合：

- GitHub 打不开
- Docker 镜像经常失败
- 公司网络限制多
- 想把安装包拷给别人

## 文件校验

正式发布时，每个下载文件都会配套 `checksums.txt`。下载后可以校验：

```bash
shasum -a 256 opendeepseek-cn-v0.5.0-cn-macos-arm64.zip
```

输出里的哈希值应和 `checksums.txt` 对应行一致。不一致就不要运行，重新下载。

## 安全边界

默认中国版只让 Agent 访问：

- `~/OpenDeepSeek-Agent/OpenDeepSeek-Inputs`
- `~/OpenDeepSeek-Agent/OpenDeepSeek-Outputs`
- `~/OpenDeepSeek-Agent/OpenDeepSeek-Memory`

如果你想让它读取桌面、下载或文档目录，需要在后续引导页里明确授权。不要把 OpenDeepSeek 暴露到公网；公网部署必须先开启登录、HTTPS 和访问控制。
