# OpenDeepSeek 小白使用手册

> 你好呀！这是你的朋友写给你的说明书。不用紧张，看完就能和 AI 聊天了 😊

---

## 第一章：30 秒上手（最简版）

**就四步，真的：**

1. **打开浏览器** —— 你平时看网页用的那个，Chrome、Safari、Edge 都行
2. **地址栏输入** `http://localhost:3000` 然后按回车
3. **看到中间的对话框** → 输入"你好" → 点右下角的发送按钮（或按回车）
4. **AI 回复你了** → 恭喜你，成功了！

> 💡 小提示：这个地址 `localhost` 意思是"你自己电脑上的网站"，不用联网也能用（只要之前装好了）。

## 第三章：5 个最常用功能怎么用

### 1. 聊天（基础对话）

**在哪点：** 首页中间的大白框就是

**怎么操作：** 打字 → 点发送（或按回车）→ 等 AI 回复

**能做什么：** 问问题、写文章、翻译、聊天、头脑风暴……就像和一个知识渊博的朋友微信聊天

> 小技巧：说得越具体，AI 回答越好。比如不要问"写首诗"，要问"写一首关于春天雨天的五言绝句"

---

### 2. 上传 PDF/Word 让 AI 读

**在哪点：** 对话框左下角的 **回形针图标 📎**（或直接把文件拖进对话框）

**怎么操作：** 点回形针 → 选电脑里的文件 → 等上传完成 → 输入你的问题，比如"这篇文档讲了什么" → 发送

**能做什么：** 
- 让 AI 总结 100 页的报告
- 问合同里的具体条款
- 对比两份文件的不同
- 从论文里找某个数据

> ⚠️ 注意：上传后 AI 是"临时读"，想让它永远记住，要放到"知识库"里（见第二章表格）

---

### 3. 联网搜资料

**在哪点：** 对话框下面的 **小地球图标 🌐**（或设置里的 Web Search 开关）

**怎么操作：** 打开联网开关 → 正常提问，比如"今天北京天气怎么样" → AI 会自动去搜

**能做什么：** 查新闻、查股价、查天气、查任何 AI 脑子里没有的最新信息

> 💡 对比：关掉联网，AI 只用它训练时学到的知识（可能到 2024 年初）；打开联网，能查今天的消息

---

### 4. 让 AI 跑 Python 代码

**在哪点：** 设置 → Code Interpreter → 打开开关；然后对话时 AI 会自动调用

**怎么操作：** 直接说"帮我算一下""画个图""分析这组数据"，AI 觉得需要写代码时会自动写、自动跑

**能做什么：**
- 复杂数学计算（比如房贷利息）
- 画图表（柱状图、折线图）
- 分析 Excel 数据
- 处理图片（裁剪、转格式）

> 你不需要会写代码，AI 全包了，你看结果就行

---

### 5. 创建多个对话

**在哪点：** 左上角 **"New Chat"**（或 + 号）

**怎么操作：** 点 New Chat → 新窗口开了 → 左边列表能看到所有对话标题

**能做什么：** 
- 一个对话聊工作，一个聊生活，不混
- 一个用中文，一个用英文
- 一个开联网，一个不开

> 左边列表点一下就能切换，不用怕开太多

## 第四章：进阶玩法（有兴趣再试）

### 收藏好用的提示词
- 某次问法效果很好？点对话里的 **⭐ 收藏** → 去 Prompts 里改名保存
- 下次直接调用，不用重新打字

### 导出对话记录（备份）
- 对话右上角 **⋯ → Export** → 选格式（推荐 Markdown 或 PDF）
- 存到电脑里，以后随时看

### 切换不同 AI 模型
- 对话框顶部或左上角，点当前模型名字（比如 deepseek-v4-flash）
- 弹出 Models 列表，选别的"大脑"
- **建议新手：** 用默认的 deepseek-v4-flash 就行，别的可能更快或更慢，但不一定更好

### 上传图片让 AI 看
- 和传文件一样，把图片拖进对话框
- 问"这张图里有什么""这张表格数据是多少""帮我描述这张图"
- OpenDeepSeek 会先用 Hermes Smart Bridge 把图片保存到 `/host/OpenDeepSeek-Inputs` 并 OCR 成文字，再按任务类型交给 DeepSeek V4 Flash 或 Hermes Agent 处理；这样既能用图片证据，又不会把 `image_url` 直接塞给 DeepSeek 文本端点。

## 第五章："我点错了在哪"急救指南 🚑

| 你看到了什么 | 这是哪里 | 怎么回去 |
|:---|:---|:---|
| 一堆英文菜单，左边有 Users、Models、Connections…… | **Admin Panel（管理员后台）** | 点左上角 **OpenDeepSeek logo** 或按键盘 `ESC` |
| 很多管道图、节点、连线 | **Pipelines（AI 流水线）** | 直接关标签页，重新开 `localhost:3000` |
| 模型列表，有 deepseek-v4-flash、llama、qwen 等 | **Models（选 AI 大脑）** | 选好点确认，或点 logo 回首页 |
| 代码编辑器一样的界面，有 function、async def | **Functions（扩展功能）** | 关！这是给程序员用的 |
| 不知道自己在哪，界面很陌生 | — | **万能方法：点左上角 logo，或刷新页面重新进** |
