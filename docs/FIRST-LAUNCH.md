## 3. 看不到我刚才说的样子？常见情况

### 情况 A：页面是英文的 🔤

**现象：** 按钮、菜单全是英文。

**解决（2 选 1）：**

**方法 1 - 网页里改：**
1. 点右上角 **👤 头像**
2. 点 **Settings**
3. 左边选 **General**
4. 找到 **Language** → 下拉选 **中文 (Chinese)**
5. 刷新页面

**方法 2 - 命令行改（彻底）：**
```bash
# 先关掉
docker compose down

# 确认 .env 里有这一行（没有就加上）
echo "DEFAULT_LOCALE=zh-CN" >> .env

# 再启动
docker compose up -d
```

---

### 情况 B：要求注册账号 / 登录 🔒

**现象：** 出现"登录""注册""Sign in"页面。

**原因：** 你的 `.env` 里 `WEBUI_AUTH=true`，这是团队模式。

**想关掉登录（变家庭模式）：**

```bash
# 1. 编辑 .env 文件
nano .env

# 找到 WEBUI_AUTH=true，改成 false
# 按 Ctrl+O 保存，Ctrl+X 退出

# 2. 彻底重启（⚠️ 会清空历史聊天记录）
docker compose down -v
docker compose up -d
```

> 💡 **注意：** `down -v` 会删除所有历史对话。想保留历史，先导出：设置 → 聊天 → 导出。

---

### 情况 C：发了消息后报错"API 调用失败" ❌

**现象：** 红色提示框，或 AI 回复"调用失败""Error"。

**最可能原因：** `.env` 里的 `DEEPSEEK_API_KEY` 错了或过期了。

**解决：**

```bash
# 1. 去 DeepSeek 官网重新复制 Key
#    浏览器打开：https://platform.deepseek.com
#    登录 → API Keys → 复制（一串 sk- 开头的字符）

# 2. 改本地文件
nano .env
# 找到 DEEPSEEK_API_KEY=... 那一行，把旧的换成新的

# 3. 重启生效
docker compose restart
```

---

### 情况 D：模型选择器是空的 / 显示 "No models found" 🔍

**现象：** `deepseek-v4-flash ▼` 那个位置显示空白或报错。

**原因：** Open WebUI 启动后第一次连 DeepSeek 服务器拉模型列表，可能慢或失败。

**解决：**

1. **先刷新页面**（按 F5 或 Ctrl+R）
2. 等 10 秒，看模型是否出现
3. 还不行，执行：
   ```bash
   docker compose restart open-webui
   ```
4. 等 30 秒，再刷新浏览器

---

### 情况 E：界面卡 / 转圈不停 ⏳

**现象：** 发送后一直转圈，或页面白屏。

**解决（3 个办法从简到繁）：**

| 操作 | 快捷键 |
|:---|:---|
| 强制刷新（清缓存） | **Windows:** `Ctrl + Shift + R` <br> **Mac:** `Cmd + Shift + R` |
| 开无痕窗口再试 | `Ctrl + Shift + N`（Chrome） |
| 重启整个服务 | `docker compose down && docker compose up -d` |

## 4. 第一次该试什么功能（按优先级）

### ⭐ 必试：基础对话（开箱即用）
- 直接打字 → 看回复
- 这就是 80% 的用法

---

### ⭐ 推荐试：让 AI 读你的 PDF 📄

**步骤：**

1. 在输入框右侧点 **📎 回形针图标**
2. 选一份电脑上的 PDF（中文也行）
3. 等 1-2 分钟（后台在识别文字、建立索引）
4. 然后输入：`总结这个文档的主要内容`

**适合：** 读论文、读合同、读说明书，不用自己逐页看。

---

### ⭐ 推荐试：联网搜资料 🌐

**前提：** 你启动时用了完整模式
```bash
docker compose --profile full up -d
```

**步骤：**

1. 在对话框上方找到 **⚙️ 设置图标** 或 **🌐 网络搜索开关**
2. 打开"网络搜索"
3. 输入：`今天上海天气怎么样`

**适合：** 查实时信息（股价、新闻、天气），弥补 AI 知识截止日期的问题。

---

### 🔧 进阶：让 AI 跑代码 ⚡

**直接输入：**
```
用 Python 画一个简单的折线图，数据是 [1, 3, 2, 5, 4]
```

**你会看到：**
- AI 生成 Python 代码
- 自动执行（在浏览器里，不用装 Python）
- 下面直接显示出折线图

**适合：** 快速验证代码、做数据可视化、学编程。

## 5. 收藏好用的 Prompt 💾

很多场景的 Prompt 已经写好了，在 **[PROMPT-COOKBOOK.md](PROMPT-COOKBOOK.md)**，复制就能用。

**怎么收藏到 Open WebUI 里：**

```
步骤：
1. 打开 PROMPT-COOKBOOK.md，找到想要的 Prompt（比如"写周报"）
2. 复制那段文字
3. 回到 OpenDeepSeek 网页
4. 左侧菜单 → 🏢 Workspace（工作区） → ✏️ Prompts
5. 点 ➕ 新建
6. 粘贴 Prompt → 起个名字，比如"周报"
7. 保存
```

**以后用：**
- 在聊天框输入 `/周报`
- 自动调出那段 Prompt，不用每次复制

## 6. 想让别人也能用？👥

你的电脑跑着 OpenDeepSeek，家人/同事想用：

### 同一 WiFi 下（局域网）

**他们的浏览器输入：**
```
http://你的电脑IP:3000
```

**怎么知道你的 IP：**
- Windows：按 `Win+R`，输入 `cmd`，执行 `ipconfig`，找"IPv4 地址"
- Mac：系统设置 → 网络 → 详细信息

**但默认只能你自己用，要改设置：**

```bash
# 1. 改 .env
nano .env
# 找到 BIND_HOST=127.0.0.1
# 改成 BIND_HOST=0.0.0.0

# 2. 重启
docker compose down
docker compose up -d
```

> ⚠️ **警告：** 改成 `0.0.0.0` 后，局域网任何人都能访问。强烈建议同时启用账号：
> ```bash
> # .env 里改
> WEBUI_AUTH=true
> ```
> 这样别人需要注册账号才能用。

---

### 远程访问（最推荐 Tailscale）

装 [Tailscale](https://tailscale.com)（免费），你的电脑会获得一个固定地址，比如 `http://your-pc:3000`，手机/公司电脑都能安全访问，不用改防火墙。
