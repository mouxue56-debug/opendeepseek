# 中文样例 Prompt 集（复制即用）

> 这些 prompt 都是中国普通用户最常用的场景，复制粘贴到对话框就能用。
> 看到 [方括号] 的地方替换成你自己的内容。

## 怎么用

使用很简单：① 复制 prompt ② 粘贴到对话框 ③ 替换 [...] 为你的内容 ④ 发送。也可以保存到 Open WebUI 的 Prompts（提示词模板）方便以后用，下次输入 `/模板名` 就能调出来。

## 工作类（5 个）

### 1. 写周报

**用途**：每周五要交周报，懒得回想这周做了什么

```
你是一个职场写作助手。我这周做的事情是：
[列出几条这周做的事]

请帮我写一份周报，结构：本周完成、下周计划、遇到的问题、需要协助的地方。语气专业但不官腔，500 字以内。
```

**期望输出**：一份格式整齐、可直接交的周报

---

### 2. 写商务邮件

**用途**：要发邮件但不知道怎么写得体

```
我要给 [收件人身份：客户 / 领导 / 同事] 发一封邮件，主题是 [邮件主题]。

邮件背景：[简单说明情况，比如"之前约定的报价"或"项目延期通知"]

我的态度是：[客气 / 正式 / 友好 / 紧急]

请写一封得体的中文邮件，200 字以内，要点清晰。
```

**期望输出**：一封可以直接发送的商务邮件

---

### 3. 总结会议纪要

**用途**：会议记录零零散散，需要整理成正式纪要

```
我参加了一场会议，以下是零散记录：
[粘贴会议记录]

请帮我整理成会议纪要，包括：会议主题、时间地点、参会人员、讨论要点、决策事项、下一步行动及负责人。
```

**期望输出**：结构清晰的会议纪要文档

---

### 4. 写 OKR / KPI

**用途**：新季度要定目标，不知道怎么写 OKR

```
我的岗位是 [岗位名称]，本季度方向是 [比如"提升用户留存"或"拓展新客户"]。

请帮我写 3 个 O（Objective），每个 O 对应 2-3 个 KR（Key Result）。要求：目标有挑战性，KR 可衡量、有时限。
```

**期望输出**：可直接提交的目标与关键结果

---

### 5. 写工作复盘

**用途**：项目做完了，需要做复盘总结

```
我刚完成了 [项目名称]，结果：[成功 / 失败 / 部分达成]

项目背景：[简单描述]

请用 5W2H 方法帮我做复盘：What（做了什么）、Why（为什么这么做）、Who（谁参与）、When（时间节点）、Where（哪里）、How（怎么做的）、How much（花了多少）。最后给 2-3 条改进建议。
```

**期望输出**：完整的复盘分析文档

---

## 学习类（3 个）

### 6. 解释复杂概念（费曼学习法）

**用途**：看到不懂的术语，让 AI 用最简单的方式解释

```
请用一个 5 岁小孩能听懂的方式，解释什么是 [量子纠缠 / 区块链 / 大语言模型 / 机器学习]。
要求：用日常生活的比喻，不超过 200 字。
```

**期望输出**：通俗易懂的解释，比喻生动有趣

---

### 7. 中英文翻译（带润色）

**用途**：不只是翻译，还要翻译得地道

```
请把以下中文翻译成地道的英文（适合 [商务 / 日常聊天 / 学术论文]）：
[粘贴中文内容]

要求：表达自然，不生硬，符合英语母语者的表达习惯。
```

**期望输出**：地道流畅的英文译文

---

### 8. 总结长文章

**用途**：文章太长没时间看，需要快速了解核心内容

```
请帮我总结下面这篇文章，输出格式：
- 一句话核心观点
- 3 个关键要点
- 我的收获/启发是什么

以下是文章内容：
[粘贴文章]
```

**期望输出**：简洁有力的文章摘要

## 生活类（3 个）

### 9. 礼物推荐

**用途**：朋友/家人生日不知道送什么

```
我要给 [关系：男朋友 / 妈妈 / 7 岁的侄女 / 领导] 选 [生日 / 节日 / 纪念日] 礼物。
对方喜好：[比如"喜欢烹饪"或"爱运动"或"实用主义者"]
预算：[X] 元
请推荐 5 个具体的礼物，每个写清楚特点、为什么适合对方、购买建议。
```

**期望输出**：5 个具体可操作的礼物方案

---

### 10. 旅游攻略

**用途**：准备出行但不知道怎么做计划

```
我计划去 [目的地：云南 / 日本 / 三亚]，时间是 [X] 天 [X] 人。

偏好是：[比如"亲子游 / 拍照打卡 / 休闲度假 / 穷游"]
体力情况：[比如"带老人孩子 / 年轻人暴走" / "每天走 2 万步没问题"]

请给我一份详细行程：每天去哪里、怎么安排、必吃美食、注意事项。总预算控制在 [X] 元以内。
```

**期望输出**：每天行程安排清晰的具体攻略

---

### 11. 食谱推荐

**用途**：不知道今天吃什么，或者想用现有食材做饭

```
我有以下食材：[比如"鸡翅、土豆、番茄、鸡蛋"]

请推荐 2-3 道菜的做法，每道包括：菜名、难度（简单/中等）、大约时间、步骤（不超过 8 步）、小贴士。
或者你也可以推荐一道家常菜。
```

**期望输出**：详细的家常菜做法

## 创作类（2 个）

### 12. 小红书文案

**用途**：发笔记没思路，想不出好标题和内容

```
帮我写一篇小红书笔记：
主题：[比如"我今天去探店了一家咖啡店"或"新买的空气炸锅开箱"]

风格要求：[文艺清新 / 接地气实用 / 专业测评]，500 字以内

要求：
- 开头要有钩子，让人忍不住想点进来
- 中间 3-5 个小段落，每段配合适的 emoji
- 结尾要有互动问题，引导评论

请直接给我文案和标题建议。
```

**期望输出**：可以直接复制发布的完整笔记

---

### 13. 公众号开头三句话

**用途**：最难的公众号开头总是写不出来

```
我要写一篇公众号文章，主题是 [比如"职场沟通技巧"或"如何存钱"]。

目标读者是：[比如"职场新人"或"25-35 岁上班族"]

请给我 5 个不同的开头方案，每个方案包括：开头三句话、文章标题建议、适合的风格（故事型/痛点型/数据型）。

要求：开头要抓人，让人想继续读下去。
```

**期望输出**：5 个候选开头和标题

## 编程类（2 个，给会一点编程的人）

### 14. 解释报错

**用途**：跑代码报错了，但看不懂错误信息

```
我跑这段代码：
```
[粘贴代码]
```

报错信息是：
```
[粘贴报错]
```

请用大白话告诉我：
1. 错在哪里
2. 为什么报错
3. 怎么改

要求：说人话，不要用太多专业术语。
```

**期望输出**：通俗易懂的错误解释和修复方案

---

### 15. 写正则表达式

**用途**：需要匹配某种格式，但不知道怎么写正则

```
我需要匹配 [需求描述，比如"中国手机号"或"邮箱地址"或"日期格式如 2024-01-01"]。

请帮我写正则表达式，并解释每一部分的含义。
```

**期望输出**：可用的正则表达式及详细解释

## 怎么收藏到 Open WebUI

1. 在对话框点「设置」按钮
2. 进入「Prompts」（提示词模板）页面
3. 点击「新建」，把上面任意一个 prompt 粘贴进去
4. 给它起个简短的名字，比如「周报」「小红书文案」
5. 保存后，以后聊天时直接输入 `/周报` 就能调出来用

---

## 进阶：自己写好 prompt 的 4 个秘诀

**1. 明确角色**
告诉 AI 你希望它是什么身份。「你是一个有 10 年经验的销售」比「你帮我写」效果好得多。

**2. 给上下文**
说明你面对的情况。「我要给客户写邮件，他对中国供应商不太信任」比「写封邮件」效果好很多。

**3. 写清要求**
明确输出格式、字数、风格。「用表格呈现，每个不超过 20 字，语气轻松活泼」比「写清楚点」更有用。

**4. 举例子**
给 1-2 个你想要的输出样例。AI 会参考例子调整输出质量。
